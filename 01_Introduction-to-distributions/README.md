Lesson 01: Introduction to distributions
=======================

The purpose of this lesson is to introduce students to the topic of spontaneous vs. acquired mutations and to make them familiar with different statistical distributions (uniform, normal, Poisson, binomial).

## Lesson Information
### Lesson Learning Outcomes
At the end of this lesson, students will be able to...
1. Generate and visualize random samples from uniform, normal, Poisson, and binomial distributions using Python and NumPy.
2. Compare the properties of common probability distributions and explain how they model different biological processes, including mutation occurrence and cell population behavior.
3. Interpret histograms and cumulative distribution functions (CDFs) to evaluate how sample size and distribution parameters influence observed experimental outcomes.


### Cyberinfrastructure Prerequisites

Before beginning this lesson, students are expected to have the following skills
and/or completed the following modules/lessons available on the ACT-CMS Portal:

* Basic skills in Python programming and syntax
    - [Introduction to Programming for Molecular Scientists](https://act-cms.molssi.org/portal/lessons/foundational-intro-python/)

### Content Prerequisites

Before beginning this lesson, students are expected to be familiar with the
following content topics:

* Definition of mutations and effects of mutations on both the individual level and the population level.
* Statistical concepts, including probability, probability distributions, sampling distributions, and confidence intervals.
